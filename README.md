# Assignment 3 — Topic Modeling of Disneyland Reviews
---
## Overview
This project applies topic modeling to Disneyland reviews across three locations:
- California
- Paris
- Hong Kong

The goal is to identify key themes in customer feedback using unsupervised learning methods.

---

## Methods
- Student A: Latent Dirichlet Allocation (LDA)
- Student B: Non-negative Matrix Factorization (NMF)

Models were evaluated using:
- Coherence score
- Perplexity

---

## Data Source
Disneyland Reviews dataset from Kaggle:
https://www.kaggle.com/datasets/arushchillar/disneyland-reviews

---

## Notes
- Data is downloaded programmatically via Kaggle API within the notebook
- Outputs are generated in-notebook for interpretation
